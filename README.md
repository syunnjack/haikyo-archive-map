# Haikyo Archive Map

Recommended repository: `haikyo-archive-map`

Primary domain candidate: `haikyoarchive.jp`

Other domain candidates:
- haikyoarchive.jp
- haikyomap.jp
- ruinmap.jp
- showaarchive.jp

## Concept

Rights-aware archive map for closed shops, Showa buildings, ruins, memory posts and safety notes.

## Built-in growth design

- SEO/AIO/LLMO answer block, FAQ, entity-rich cards, robots.txt, sitemap.xml and llms.txt
- UGC submission desk stored in localStorage for review/correction/photo style reports
- Revenue sections for affiliate, lead generation, paid listings, alerts, reports or sponsorships
- Saved lead flow for validating commercial intent
